# nexxlytic-flowx-meta

Nexxlytic FlowX — Monorepo containing both `frontend` (React) and `backend` (Node.js/Express) for a single Vercel deployment.

## Project Structure

```
nexxlytic-flowx/
├── frontend/    # React app
└── backend/     # Node.js / Express API
```

## Getting Started

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```
